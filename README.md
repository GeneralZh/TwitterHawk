Biscuit
=======

Classifiers for SemEval 2015 Tasks.


SemEval 2015       http://alt.qcri.org/semeval2015/task10/




Installation

    Required python modules:

        - numpy
        - scikit-learn
        - scipy
        - nltk
        - nose
        - BeautifulSoup
        - twitter


    Must create a BISCUIT_DIR environment variable.
